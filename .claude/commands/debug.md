## Task

Debug this issue: $ARGUMENTS

## Process

1. **Capture**: Get the full error message, stack trace, and context
2. **Reproduce**: Understand when and how the error occurs
3. **Isolate**: Find the exact location and cause
4. **Fix**: Implement the minimal fix needed
5. **Verify**: Confirm the fix works without side effects

## Output

Provide:
- Root cause explanation
- The fix (code changes)
- How to prevent similar issues
